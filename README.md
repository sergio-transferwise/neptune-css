# Thalassa

## CSS library as part of the Neptune Design System
(POC. WIP)

### Available components:

#### New
- Skew
- Grid layout
- Flexbox layout
- Wrapping
- Background
- Spacing

#### From bootstrap
- Badge
- Button
- Header navbar


## Open the library to see the demo and documentation

Clone and run in the project directory:

- `npm install`

- `npm run build`

- `npm run start`
